Packagebe.nascom.flash.graphics
Classpublic class DistortBitmap

DistortBitmap allows us to deform a bitmap freely to any quadrilateral. Due to actionscript speed limitations, this is an approximation, and divides the bitmap into segments to which linear transformations are applied.


Example
The following code transforms a bitmap and draws it to a sprite, assuming bitmapData is a loaded bitmap, and targetSprite is already on the stage: 
    import be.nascom.flash.graphics.DistortBitmap;
    import be.nascom.flash.math.Point2D;
    
    var distortBitmap : DistortBitmap = new DistortBitmap(bitmapData, 1, true);
    var points : Array = new Array();
  
    points.push(new Point2D(10, 20));
    points.push(new Point2D(100, 150));
    points.push(new Point2D(50, 40));
    points.push(new Point2D(80, 100));
    
    distortBitmap.points = points;
    distortBitmap.draw(targetSprite);
