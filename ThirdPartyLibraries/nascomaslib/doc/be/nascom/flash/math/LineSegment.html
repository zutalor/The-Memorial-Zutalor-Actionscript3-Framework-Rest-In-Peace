Packagebe.nascom.flash.math
Classpublic class LineSegment

The LineSegment Class represents a 2D line segment from one point to another, or an infinite line through two points. This class makes use of the standard AS3 Point rather than NascomASLib's Point2D/Vector2D, to ensure compatability when these classes are not used.

slopeproperty
slope:Number  [read-only]

The slope of a line. if x1==x0, slope would be +/-Infinity, and therefore it returns the maximum/minimum Number value. However, it is recommended to use a valid work-around for strictly horizontal lines.

containsPoint()method
public function containsPoint(pointX:Number, pointY:Number, infinite:Boolean = false):Boolean

Checks if a point is on the line or line segment.

Parameters
pointX:Number — The horizontal coordinate of the point
 
pointY:Number — The vertical coordinate of the point
 
infinite:Boolean (default = false) — Defines whether the LineSegment is treated as an infinite line or a segment. If true, the start and end points of the line segment are ignored, and it is handled as an infinite algebraic line. Default value is false, so the line will be treated as a segment. Ie, the function returns false unless it falls within the bounds of both segments.
