Classpublic class FMSCameraStream
InheritanceFMSCameraStream flash.events.EventDispatcher

FMSCameraStream provides functionality to easily control recording or publishing a webcam stream using Flash Media Server or a third-party rtpm-based server (such as Wowza).

Only tested in Flex
