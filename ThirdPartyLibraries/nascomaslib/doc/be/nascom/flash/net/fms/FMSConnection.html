Packagebe.nascom.flash.net.fms
Classpublic class FMSConnection
InheritanceFMSConnection flash.events.EventDispatcher

The FMSConnection class wraps basic media server functionality into one. It automatically retreives a user shared object on the media server which keeps track of currently logged in users. This class is ideally used as a baseclass for the client-side service class.

public function FMSConnection(domain:String, application:String, instance:String = "_definst_", client:String = null, protocol:* = rtmp)

Initializes the media server connection.

Parameters
domain:String — The domain name of the server.
 
application:String — The name of the application on the media server.
 
instance:String (default = "_definst_") — The instance of the application on the media server, by default _definst_.
 
client:String (default = null) — Indicates the object on which callback methods should be invoked. The default is this FMSConnection instance. If you set the client property to another object, callback methods will be invoked on that object.
 
protocol:* (default = rtmp) — The protocol used by the media server.
