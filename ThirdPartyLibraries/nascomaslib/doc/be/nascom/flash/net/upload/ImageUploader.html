Packagebe.nascom.flash.net.upload
Classpublic class ImageUploader
InheritanceImageUploader flash.events.EventDispatcher

ImageUploader takes a BitmapData object and uploads it to a server using a serverside upload script. Currently, only JPG filetypes are possible, and the filename is defined by the serverside script. The serverside script returns the filename of the uploaded file.
