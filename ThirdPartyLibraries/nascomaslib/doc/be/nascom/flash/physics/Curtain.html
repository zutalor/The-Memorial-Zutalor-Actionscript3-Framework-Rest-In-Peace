Classpublic class Curtain
InheritanceCurtain flash.display.Sprite

The Curtain class takes a DisplayObject and uses it to mimic the movement of an opening or closing curtain.


Example
The following example code creates a Curtain instance and animates it (curtainSource is assumed to be a DisplayObject instance): 
  import be.nascom.flash.physics.Curtain;
  
  var curtain : Curtain = new Curtain(curtainSource, 7);
  addChild(curtain); 
  curtain.slide(-50, 10, Curtain.LEFT_SIDE);
  curtain.slide(-200, 10, Curtain.RIGHT_SIDE);

inertiaproperty 
inertia:Number  [read-write]

The amount of inertia of the curtain. Higher values will cause the curtain segments to react slower. It is different from friction because it affects how long it will take for each segment to react to the pull of the previous one, and not the resistance it will meet while moving.

Curtain()constructor
public function Curtain(source:DisplayObject, segments:uint = 7, interpolationDetail:uint = 5, animated:Boolean = false)

Creates a Curtain object from a target DisplayObject

Parameters
source:DisplayObject — The DisplayObject which will serve as the surface of the curtain.
 
segments:uint (default = 7) — The amount of segments out of which the curtain will exist. Higher values will look less segmented, but are slower.
 
interpolationDetail:uint (default = 5) — Amount of detail with which the segments should be interpolated.
 
animated:Boolean (default = false) — Indicates whether the source will be animated or not.

slide()method
public function slide(pixels:Number, frames:uint, side:String = "right"):void

Starts the animation of one side of the curtain, and slides over a distance of pixels.

Parameters
pixels:Number — The horizontal position in pixels of the given side of the curtain, relative to the original position. A positive value of n will animate the side by n pixels to the right. A negative value will slide to the left.
 
frames:uint — The speed in frames of the animation.
 
side:String (default = "right") — Indicates which side of the curtain to animate. Possible values:
  • Curtain.LEFT_SIDE
  • Curtain.RIGHT_SIDE
