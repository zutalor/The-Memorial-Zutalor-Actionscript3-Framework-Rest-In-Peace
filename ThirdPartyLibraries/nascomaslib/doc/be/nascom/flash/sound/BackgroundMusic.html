Packagebe.nascom.flash.sound
Classpublic class BackgroundMusic

The static BackgroundMusic class controls background music throughout a whole site. Currently, it only supports playback from a reference embedded in the library.

volumeproperty
volume:Number  [read-write]

The volume of the sound in unmuted state. Changing this value will not affect the output volume if muted, but when unmuted it will revert to the new value.
