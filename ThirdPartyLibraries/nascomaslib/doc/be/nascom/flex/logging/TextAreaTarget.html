Classpublic class TextAreaTarget
InheritanceTextAreaTarget mx.logging.targets.LineFormattedTarget

Provides a logger target that uses a text area to output log messages.


Example
The following code shows you how to create and use the FileTarget. 
  // Instance variable that contains a ref to the Logger
  private var logger:ILogger = Log.getLogger("LoggerName");
  
  ...
  
  // Create the TextAreaTarget for logging
  var textAreaTarget:TextAreaTarget = new TextAreaTarget();
  textAreaTarget.includeLevel = true;
  textAreaTarget.includeCategory = true;
  textAreaTarget.level = LogEventLevel.ALL;
  
  // Add the target to the Flex Logging framework
  Log.addTarget(textAreaTarget);
   
  // Do the actual log call
  logger.info("Logging this value: " + testVar);
