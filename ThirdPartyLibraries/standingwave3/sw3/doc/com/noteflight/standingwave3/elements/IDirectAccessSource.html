Method Detail
fill()method
public function fill(offset:Number = -1):void

Fill is called to guarantee that the sample memory is valid up until the offset specified. Used extensively by CacheFilter and any other unit generators that are direct access. If called without a param, it will attempt to fill the entire buffer.

Parameters

offset:Number (default = -1)

getSamplePointer()method 
public function getSamplePointer(frames:Number = 0):uint

Return the pointer to the sample memory so that other sample functions can read directly from sample memory. May return null if asked for a pointer to invalid memory.

Parameters

frames:Number (default = 0) — the starting offset point of the range (inclusive)

Returns
uint
useSample()method 
public function useSample(frames:Number):void

useSample is like getSample(), except it does not return a Sample. It just guarantees that the next frames are available in memory. If the source is also an IAudioSource, it will also advance the position.

Parameters

frames:Number — the starting point of the range (inclusive)
