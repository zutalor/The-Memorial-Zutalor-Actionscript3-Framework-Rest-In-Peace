Classpublic final class Sample
InheritanceSample Object
Implements IAudioSource, IRandomAccessSource, IDirectAccessSource

Sample is the fundamental audio source in StandingWave, and is the primary way audio data is passed through the application. Sample also provides access to a library of basic operations through its "Alchemical Wave" C Libraries. Every sample has a fixed-size internal sample memory allocated for it, and its data may be manipulated easily through the built in functions. Alternatively, the sample data can be requested as a Vector of Numbers, manipulated, and then committed back to the sample memory.

standardize()method 
public function standardize():void

Standardize migrates a sample with any descriptor format to 44.1k stereo. Mono signals are steroized, and 22050 Hz data is upsampled to 44100 Hz. There is no change to samples in the correct format. This is called by the AudioSampleHandler before passing anything to Sound output, but may also be used any time in the processing chain that it is needed.

useSample()method 
public function useSample(numFrames:Number):void

IDirectAccessSources can be used similarly to IAudioSource. Call useSample() instead of getSample(). This instructs the source to "fill" the next n frames, so that they will have valid data. That data is then safe to access through its sample pointer. Note that the Sample implementation is trivial/unnecessary, and this is primarily used with CacheFilter.

Parameters

numFrames:Number — the number of additional frames to use from this sample.
