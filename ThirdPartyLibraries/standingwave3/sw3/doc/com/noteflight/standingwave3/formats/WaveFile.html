Converts a ByteArray containing raw fixed point audio data into a Wave file. Works by generating a WAV header, and then copying the raw data after.

Parameters

rawDataBytes:ByteArray — the source ByteArray containing raw fixed point audio data
 
sampleRate:uint — the sampling rate of the raw audio data
 
numChannels:uint — the number of interleaved channels of audio data
 
bitDepth:uint — the bit depth of the audio data
