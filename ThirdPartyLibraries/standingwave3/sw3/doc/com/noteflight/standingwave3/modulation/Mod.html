Packagecom.noteflight.standingwave3.modulation
Classpublic class Mod
InheritanceMod Object

A Mod represents a segment that is used as a control value for some synthesis function. It contains four points so that operators can create a continuously smooth spline function through any time period, a straight line, or variations of such.
